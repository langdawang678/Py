WebDriver for Java

WebDriver环境的搭建：
1. 安装配置JAVA运行开发环境
   安装JDK、JRE
   JDK：java development kit java开发环境
   JRE：java runtime environment  Java运行环境
   配置java环境变量（参照文档配置）
   验证：在cmd中输入java和javac都正确返回

2. 安装JAVA开发环境eclipse
   解压缩eclipse在c盘根目录下，直接运行eclipse.exe， 设置workspace

3. 在eclipse中创建一个项目  java project： Selenium Project

4. 安装WebDriver
   将selenium-2.42.2解压缩，将其中所有的文件复制到新建的项目的根目录下；
   将Selenium的所有相关jar包加载到新建的项目中：
   在eclipse中项目 右键---Build Path--Configure Build Path--选择Libraries--Add External JARS 将selenium 的jar包及libs文件夹下的所有jar包都加载进来

    结果：在项目的目录下，会多一个referenced Libraries的目录，其中包含刚才加载的所有jar包

    jar包就是别人已经写好的一些类，然后将这些类进行打包，你可以将这些jar包引入到项目中，然后就可以直接使用其中的类和类的方法与属性。

 ---- Java基本概念---
类与对象
类： 是具备某些共同特征的实体的集合，它是一个抽象的概念
对象：是真实世界中的实体，对象与实体是一一对应的，是一种具体的概念
比如类：人类  对象：张三 李四

属性与方法
方法：描述对象的动作，在程序设计中可以称为函数
属性：描述对象的静态特征，在程序设计中可以称为变量

比如：方法：吃饭、唱歌、跑步等   属性：身高、体重、年龄等

命令规则：源文件名必需与public类的名相同。

WebDriver的元素定位
1. id
findElement（By.id("id属性值"));
2. name 表单定位首先
findElement（By.name("name属性值")）;

3. 超链接
findElement(By.linkText("超链接文字"));

4. CSS Locator
findElement(By.cssSelector("input[name='name属性值']"));

5. Xpath
findElement(By.xpath("//input[@name='name属性值']"))

练习：
将输入用户名与密码的步骤改成css Locator方式；
点击登录按钮与退出登录链接改成用xpath方式

----Junit4框架----
将IDE脚本转化为Java/Junit4/WebDriver的脚本

步骤：
1. 将IDE脚本用IDE打开
2. 在IDE的file中选择export test case as--选择转化的格式为Java/Junit4/WebDriver
   转化后为一个.java文件
3. 将转化后的java文件复制到项目的工作空间workspace相应项目src相应包下
4. 在eclipse中刷新后，即可看到转化后的java文件
